# discomap.Tools.Wiki

This folder contains the required files to generate the [Discomap WIKI page](https://eea.github.io/discomap.Wiki/) using [MKDocs](https://www.mkdocs.org/) and deployed with GitHub pages.
Edit or add the .md files on the main branch. The build and deployment is automatic, it takes a few seconds/minutes until you can see the changes live on production.
