# discomap.Tools.Wiki

This folder contains the required files to generate the [Discomap WIKI page](https://discomap.eea.europa.eu/wiki/) using [MKDocs](https://www.mkdocs.org/).

The idea is to work on the **CWS shared folder**; at S:\DiscomapAdmin\discomap-wiki.

Main commands to operate the tool:

> mkdocs serve (to preview the generated app)

> mkdocs build (to generate the app to be deployed)
