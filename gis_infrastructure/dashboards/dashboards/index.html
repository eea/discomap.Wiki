ArcGIS Dashboards

ArcGIS Dashboards is a configurable web app in ArcGIS Portal that provides a comprehensive and engaging view of your data. Dashboards display multiple visualizations that work together on a single screen.

Key Features

  • Interactive Elements: Include maps, charts, gauges, and other elements that update dynamically based on user interactions.
  • Configurable: Tailor dashboards to meet specific needs without any coding, using the interactive configuration settings.
  • Data Integration: Combine data from multiple sources, including ArcGIS layers, web services, and external APIs.

Basic Functionality

  1. Data Source Connection: Connect to your data sources, which can include feature layers, map services, or other data available in your ArcGIS Portal.
  2. Designing the Dashboard: Use the dashboard's configuration interface to drag and drop elements such as maps, charts, lists, indicators, and more.
  3. Interactivity Setup: Configure interactions between elements so that selections in one element can filter or update other elements.
  4. Sharing: Once your dashboard is configured, you can share it with specific users, groups, or the public.

Getting Started

To get started with ArcGIS Dashboards, you can follow the official documentation.

For a detailed guide on how to create and configure your dashboards, refer to the ArcGIS Dashboards Documentation.
